# Contributors

Add a small profile card here if you make a contribution.

Use `contributors/your-github-username.md`.

Keep it simple:

```md
## @your-github-username

- Learning: Git, GitHub, and open source
- First contribution: documentation improvement
- Favorite command: git status
- Goal: make one helpful contribution

Do not include private contact details such as personal phone numbers, home addresses, or personal email addresses.
