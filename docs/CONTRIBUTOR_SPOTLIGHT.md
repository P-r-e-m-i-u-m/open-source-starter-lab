# Monthly Contributor Spotlight

The monthly spotlight is a small public thank-you for contributors who make the repo more useful for beginners.

It is not based on who writes the most code. It is based on useful, reviewable work.

## What Counts

- A focused PR that closes a beginner issue.
- A second PR that improves quality, tests, docs, or workflow.
- A helpful discussion answer.
- An issue cleanup that makes a task easier for the next contributor.
- A contributor passport that shows real proof.
