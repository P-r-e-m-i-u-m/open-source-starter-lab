# Skill-Based First Issues

Pick a first issue by what you already know.

People do not usually think, "I want issue #22." They think, "I know HTML, JavaScript, Python, docs, testing, or Git. What can I do?"

Use this page to choose a first pull request that fits your current skill.

## HTML/CSS

Good for people who know basic HTML, CSS, Markdown, or page layout.

- [HTML/CSS first issues](https://github.com/P-r-e-m-i-u-m/open-source-starter-lab/issues?q=is%3Aissue+is%3Aopen+label%3A%22skill%3A+html-css%22)
- Start with: improving readable sections, Markdown tables, guide layout, or contributor-facing docs.

## JavaScript

Good for people who know JavaScript or want to start reading TypeScript.

- [JavaScript first issues](https://github.com/P-r-e-m-i-u-m/open-source-starter-lab/issues?q=is%3Aissue+is%3Aopen+label%3A%22skill%3A+javascript%22)
- Start with: CLI output, small command behavior, examples, or TypeScript source files.

## Python

Good for people who know Python first and want to learn GitHub contribution workflow.

- [Python beginner issues](https://github.com/P-r-e-m-i-u-m/open-source-starter-lab/issues?q=is%3Aissue+is%3Aopen+label%3A%22skill%3A+python%22)
- Start with: docs that explain how Python learners can contribute to a TypeScript repo.

## Docs

Good for people who like writing, explaining, teaching, or fixing unclear steps.

- [Docs first issues](https://github.com/P-r-e-m-i-u-m/open-source-starter-lab/issues?q=is%3Aissue+is%3Aopen+label%3A%22skill%3A+docs%22)
- Start with: beginner guides, setup instructions, Git examples, or PR description examples.

## Testing

Good for people who want to make a real code contribution without changing product behavior.

- [Testing first issues](https://github.com/P-r-e-m-i-u-m/open-source-starter-lab/issues?q=is%3Aissue+is%3Aopen+label%3A%22skill%3A+testing%22)
- Start with: smoke tests, CLI behavior checks, invalid input cases, or test documentation.

## Git

Good for people who are learning branches, commits, push, pull requests, and CI.

- [Git beginner issues](https://github.com/P-r-e-m-i-u-m/open-source-starter-lab/issues?q=is%3Aissue+is%3Aopen+label%3A%22skill%3A+git%22)
- Start with: safe command examples, first PR workflow guides, and common Git errors.
