## How To Claim One

Comment on the issue:

```md
I would like to work on this. Please assign me.

I will run `npm run check` and keep the PR focused.
```

## How To Choose

Pick the issue where you can answer these questions:

- What file should I open first?
- What command should I run first?
- What should I avoid changing?
- What proof should I put in the PR?

If you cannot answer those yet, use the [First Issue Decoder](ISSUE_DECODER.md) or ask in the [weekly assignment thread](https://github.com/P-r-e-m-i-u-m/open-source-starter-lab/discussions/44).
