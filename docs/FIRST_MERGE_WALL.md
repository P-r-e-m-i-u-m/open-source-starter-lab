# First Merge Wall

This wall celebrates contributors who made a real pull request, got review feedback, and reached a clean merge.

If this repo helped you make your first pull request, first open-source contribution, or first GitHub merge, add your entry here in a small follow-up PR.

## How To Add Yourself

1. Open a new branch.
2. Add your entry to the wall.
3. Link your merged PR.
4. Run `npm run check`.
5. Open a pull request.

Keep your entry short and professional.

## Entry Template

```md
### @your-github-username

- First merged PR: #123
- I worked on: docs, CLI, tests, or contributor help
- I learned: one short thing you learned
```

## Why This Exists

Open source can feel invisible when you are starting out. This page gives new contributors a simple record of their first merge and shows future visitors that real beginners are welcome here.
