# Open Source Starter Lab Website

This is a static web hub for the repo.

Open locally:

```text
site/index.html
```

The site has no build step and no runtime dependencies. It is designed to be easy to publish with GitHub Pages later.

## Sections

- Hero command center
- Interactive issue finder
- Contributor journey
- First PR workflow
- Live proof links
- Trust signals
- Maintainer automation showcase
- Beginner FAQ
