# Contributing Guide

## Welcome
Thank you for contributing to the Open Source Starter Lab!

This guide follows best practices from Meta, Google, and other top open source projects.

## How to Contribute
1. Check open issues labeled 'good first issue'.
2. Comment to get assigned.
3. Fork & clone.
4. Create feature branch.
5. Make small changes.
6. Test with `npm run check`.
7. Submit PR with clear description.

## PR Checklist
- [ ] Tests pass
- [ ] Follows code style
- [ ] Links to issue

Questions? Open a Discussion.
