# Code of Conduct

## Our Standard

This project is for learning. Everyone should feel safe asking beginner questions and making small improvements.

Use respectful language, assume good intent, and explain corrections clearly.

## Not Okay

- Harassment or insults
- Spam
- Fake contribution activity
- Asking people to accept answers only for badges
- Sharing private information

## Maintainer Response

Maintainers may edit, hide, or remove content that harms the learning environment.
